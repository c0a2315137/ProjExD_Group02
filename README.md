# 生きろこうかとん！

## 実行環境の必要条件
* python >= 3.13.2
* pygame >= 2.6.1

## ゲームの概要
* 主人公キャラクターこうかとんをキーボード操作で動かし敵を倒す

## ゲームの遊び方
* 矢印キーでこうかとんを操作し，スペースキー押下によるビームで敵を倒す
* スコアによってレベルアップし、機能が解放されていく

## ゲームの実装

### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能
* 1.〇〇(C0A2....):経験値とレベルアップ
* 2.:武器システム
* 3.:敵に関して
* 4.:アイテム・ドロップ
* 5.:UI関連
